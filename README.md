# 🛡️ 腦錢包離線保險箱 (Brainwallet Offline)

這是一個基於 **確定性演算法 (Deterministic Algorithm)** 的密碼生成工具。它不需要資料庫、不需要連網，完全透過「你的腦袋」搭配「數學運算」來找回密碼。

## 🌟 核心特色

- **100% 本地運算**：所有加密過程皆在瀏覽器內完成（WebCrypto API），絕不回傳伺服器。
- **PWA 支援**：可安裝至手機主畫面，支援 **完全離線使用**（飛航模式也沒問題）。
- **中文友善**：支援中文主密碼與鹽值輸入，搭配「智慧輸入模式」，點擊自動顯示明碼。
- **多重加鹽 (Salting)**：不同長度與格式的密碼皆經過獨立雜湊處理，避免資訊洩漏。
- **隱私遮蔽**：生成結果預設遮蔽中間段，僅在點擊眼睛圖示時暫時顯示。

## 🚀 如何使用

1. **服務名稱**：輸入該帳號的標籤（例如：`google`、`玉山銀行`、`Steam`）。
2. **腦錢包秘密**：輸入你腦中唯一的「主金鑰」。
3. **自定義鹽值**：(選填) 輸入額外的秘密字串（如：`2026`）來讓密碼更隨機。
4. **生成與複製**：點擊按鈕後，選擇適合該網站長度的密碼點擊 📋 即可。

## 🔒 安全性說明 (Security Principles)

本工具遵循 **柯克霍夫原則 (Kerckhoffs's Principle)**：
> 系統的安全性不應依賴於演算法的保密，而應僅依賴於金鑰 (你的腦錢包) 的保密。

即使本專案的原始碼完全公開，只要你的「主密碼」與「鹽值」組合夠複雜，目前最頂尖的超級電腦也無法在有效時間內暴力破解出你的密碼。

## 📱 安裝至手機

1. 使用手機瀏覽器開啟網頁。
2. **iOS (Safari)**：點擊「分享」>「加入主畫面」。
3. **Android (Chrome)**：點擊選單 >「安裝應用程式」或「加到主畫面」。

### ⚠️ 免責聲明
本工具僅提供加密運算服務。請務必記牢你的「主密碼」與「鹽值」，若因遺忘而導致無法產生相同密碼，開發者與本工具無法協助找回。
